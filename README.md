# 프로젝트 개요

- github project기능의 클론 코딩인 to-do list 프로젝트입니다.

# 팀 룰

### 협업 방식

- 페어 프로그래밍(설계)
- 분업(구현)
- 코드 리뷰는 로컬에서 기능이 완성되어 외부 저장소에 pull, merge하기전에 하기

### 해보고 싶은 거

- 스켈레톤 코드 및 설계
- 상세한 단위일정
- 테스트 코드 작성

### 그라운드 룰

- 깃 커밋 스타일 가이드 : https://siyoon210.tistory.com/56
- 코드 스타일 가이드 : airbnb eslint + prettier
- 업무와 휴식 비율은 6 : 1
- 모닝커피는 팀원과

### 욕심사항

- 예쁜 디자인으로 UI변경
